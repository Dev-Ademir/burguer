# 🍔 Hamburgueria Online

Cardápio digital interativo para hamburgueria, desenvolvido como projeto de estudo em transição de carreira para Desenvolvimento Full Stack.

## 🎯 Sobre o Projeto

Aplicação frontend que simula um cardápio online completo, com exibição de produtos, preços e interface responsiva. Desenvolvido com foco em usabilidade e design intuitivo.

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (Flexbox, responsividade)
- JavaScript vanilla (ES6+)
- Manipulação do DOM

## ✨ Funcionalidades

- [x] Exibição de cardápio com imagens
- [x] Listagem de produtos com preços
- [x] Design responsivo para mobile
- [x] Interface intuitiva e visual atrativo
